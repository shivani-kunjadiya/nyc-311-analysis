# NYC 311 Calls Analysis Using Hive

This project analyzes the **New York City 311 service request dataset** using **Apache Hive**, a data warehouse tool built on top of Hadoop. The dataset contains a wide variety of complaints and service requests submitted by residents across New York City.

## Project Goal

The main objective of this project is to:

- Load the dataset into **Hive**  
- Run queries to explore and analyze the data  
- Identify common issues reported by residents  
- Extract patterns and meaningful insights  

The analysis aims to provide actionable insights that can help the city improve its services and address residents' concerns more effectively.

## Final Report Highlights

The report includes:

- **Key findings** from the dataset  
- **Clear insights** based on complaint categories  
- **Suggestions and recommendations** for the Mayor of New York to improve city services  

## Visualization

To communicate the results effectively, the project uses:

- Tables to summarize data  
- Bar charts and pie charts to highlight trends  
- Visualizations to support recommendations and explain actionable steps  

These visuals help illustrate the insights clearly and make it easier to understand which actions are needed to enhance the quality of life for New Yorkers.
